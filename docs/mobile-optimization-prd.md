## 점검 대상 페이지

### 1. 메인 페이지 (리포트 목록)
- URL: `/?client=hea-pangyo`
- 점검 항목:
  - [ ] 리포트 카드 레이아웃 (1열 배치)
  - [ ] 터치 타겟 크기 (최소 44px)
  - [ ] 텍스트 가독성
  - [ ] 필터/검색 UI

### 2. 리포트 상세 페이지
- URL: `/report/monthly/[id]`
- 점검 항목:
  - [ ] 헤더 영역 (클라이언트명, 기간)
  - [ ] KPI 카드 (2x2 그리드 → 1열 또는 2열)
  - [ ] 차트 (가로 스크롤 또는 축소)
  - [ ] 테이블 (가로 스크롤)
  - [ ] AI 인사이트 섹션
  - [ ] 하단 푸터

## 점검 체크리스트

### 레이아웃
- [ ] 가로 스크롤 없이 콘텐츠가 화면에 맞는가
- [ ] 패딩/마진이 적절한가 (너무 좁거나 넓지 않은가)
- [ ] 그리드가 1~2열로 적절히 변환되는가

### 타이포그래피
- [ ] 텍스트 크기가 읽기 편한가 (최소 14px)
- [ ] 줄 간격이 적절한가
- [ ] 긴 텍스트가 잘리거나 넘치지 않는가

### 터치 인터랙션
- [ ] 버튼/링크 터치 영역이 충분한가 (최소 44x44px)
- [ ] 인접한 터치 요소 간격이 충분한가 (최소 8px)
- [ ] 스크롤이 부드러운가

### 차트/테이블
- [ ] 차트가 화면에 맞게 축소되는가
- [ ] 테이블이 가로 스크롤 가능한가
- [ ] 숫자가 잘리지 않는가

### 성능
- [ ] 이미지 최적화
- [ ] 불필요한 JS 로드 최소화
- [ ] First Contentful Paint < 2초
